Projects

これまでに主導・参画した主なプロジェクトや制作物です．

農業DX / センサーネットワーク

でぃー農X / D-Farmer

農家の後継者不足やアナログな作業環境といった課題を解決する統合型農業DXソリューションです．主にトマトを対象とし，カメラ画像からYOLOやMobileNetV2を用いて糖度・酸度・大きさを推定するAIモデルを構築しました．さらに，GNSSに依存しないBLEビーコンを用いた屋内位置情報取得技術を組み合わせ，スマートフォンやXRゴーグル上で作業員へリアルタイムに生育状況を提示するシステムを開発しています．

Python AI (MobileNetV2/YOLO) BLE Beacon
屋内測位 / XRアプリケーション

迷わなたす

複雑化する屋内施設での迷子問題と，施設管理のDX化を同時に解決する屋内位置情報取得システムです．BLEビーコンの受信電力を用いた三点測位方式（フリスの伝達公式を応用）により高精度な自己位置推定を実現しました．私はXR/サーバーエンジニアとして，施設情報を一元管理するAPIサーバーの構築と，HoloLens2を用いて空間上に清掃履歴や設備情報を重畳表示するシステムの実装を担当しました．

HoloLens2 三点測位アルゴリズム REST API
生成AI / Webアプリケーション

AI選書サービス「ぴっく君」

若者の活字離れに対し，「音楽」を切り口として本と出会う体験を提供するAI選書サービスです．あらすじだけでなく「読み味（クオリア）」で本と人をマッチングさせるため，大規模言語モデル（Qwen）に独自の教師データをFine-Tuningさせました．ユーザーが入力した楽曲のテーマや情景描写を分析し，親和性の高い書籍を提案するバックエンド（FastAPI）とローカルGPU環境の構築を担当しました．

LLM (Qwen) Fine-Tuning FastAPI
